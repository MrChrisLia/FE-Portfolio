# Christopher Lia — Security Engineer Portfolio

Personal portfolio site showcasing my certifications, experience, and skills as an OSCP-certified security engineer. Hosted on GitHub Pages.

🔗 **Live site:** [MrChrisLia.github.io/FE-Portfolio](https://mrchrislia.github.io/FE-Portfolio)

---

## About

Static site built with plain HTML, CSS, and JavaScript — no build step, no frameworks. Designed with a cyberpunk aesthetic (animated red → purple → blue background) and glassmorphism cards.

## Features

- Trilingual support (English / 中文 / 日本語) with live language switching
- Infinite auto-scrolling carousels for Skills and Certifications
- Accordion experience panels with translated content
- Responsive layout (mobile-friendly)

## Stack

- HTML5 / CSS3 / Vanilla JS
- Bootstrap 5.3 (navbar collapse)
- Font Awesome 6.5 (icons)
- Google Fonts — Space Grotesk, Inter, JetBrains Mono
